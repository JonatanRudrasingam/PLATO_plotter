An unoffcial tool for viewing stars in the PLATO LOPs.
\n
List of files:
\n
Tutorial.inpynb: A Jupyter notebook tutorial for plotting PLATO LOPs with a list of given targets
\n
lops2_healpix_9_footprint.fits: A FITS file containing the PLATO LOP
\n
plato_plotter.py: Python script containing the main plotting function
